# Chaitanya Sarees

A modern saree catalog and storefront experience built with Next.js and TypeScript.

## Highlights

- Product-focused frontend structure
- Next.js app architecture
- Reusable UI components
- Middleware-ready application setup
- Organized TypeScript project layout

## Tech Stack

- Next.js
- TypeScript
- React
- Tailwind/PostCSS

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Purpose

This project demonstrates a clean web experience for a fashion/saree business, with room for catalog, product, and e-commerce features.
